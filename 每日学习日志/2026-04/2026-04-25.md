2.  **关键发现：浮点数计算精度误差**：
    *   在测试绕Z轴旋转90度时，理论结果应为`(0, 1, 0)`，实际输出为`(7.54979e-08, 1, 0)`。
    *   **原因分析**：此非程序错误，而是**浮点数计算的本质局限**。计算机无法精确表示π，`cos(π/2)` 无法得到精确的0，而是一个极接近0的近似值。这是所有数值计算程序都需要面对和处理的典型问题。
    *   **解决方案**：在图形学及科学计算中，比较浮点数相等必须使用**容差(Epsilon)比较**，而非精确相等。

## 三、插值算法理论学习（新增独立模块）

今天下午独立学习了插值算法的核心理论，重点掌握了以下内容：

### 1. 线性插值（Linear Interpolation, Lerp）的理论掌握
**核心原理**：在两个已知数据点 `(x₁, y₁)` 和 `(x₂, y₂)` 之间，假设数值呈线性变化，通过比例参数 `t` 估算中间任意点的值。

**数学本质**：
```
插值结果 = 起点值 + 比例因子 × (终点值 - 起点值)
```
其中比例因子 `t = (x - x₁) / (x₂ - x₁)`，满足 `0 ≤ t ≤ 1`。

**理解深化**：
- 认识到线性插值本质是**加权平均**：`结果 = (1-t)·y₁ + t·y₂`
- 明确区分了**插值对象是数值**（如颜色强度、坐标分量、物理量等），而非位置本身
- 理解其**连续性特性**：在线段内部是C⁰连续（函数值连续）

**与图形学的潜在联系**（尚未实现，仅为理论认识）：
- 可用于颜色渐变、动画关键帧过渡、顶点属性插值等
- 是后续学习更复杂插值（如样条）的基础

### 2. 双线性插值（Bilinear Interpolation）的理论突破
**从一维到二维的扩展逻辑**：
在矩形区域的四个已知角点值 `f(Q₁₁)`, `f(Q₁₂)`, `f(Q₂₁)`, `f(Q₂₂)` 之间，通过**两次线性插值**估算内部任意点 `P(x, y)` 的值。

**两步插值过程理解**：
1. **水平方向两次插值**（在y固定时）：
   - 在顶部边沿：`R₁ = Lerp(f(Q₁₁), f(Q₁₂), x方向比例)`
   - 在底部边沿：`R₂ = Lerp(f(Q₂₁), f(Q₂₂), x方向比例)`
2. **垂直方向一次插值**（在x固定时）：
   - `f(P) = Lerp(R₁, R₂, y方向比例)`

**关键理论认知**：
1. **输入输出明确**：输入是四个角点的**坐标**和对应的**数值**，输出是内部点的**估算值**
2. **插值顺序无关性**：先水平后垂直，或先垂直后水平，数学结果相同
3. **应用场景理论对应**：
   - **图像缩放**：为放大图像中新像素计算颜色值
   - **纹理映射**：解决纹理坐标不精确对应纹素时的平滑采样问题
   - **网格数据重建**：在规则二维网格（如高度场、温度场）中估算任意位置数值
