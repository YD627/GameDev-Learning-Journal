## 转换(Transformation)
1. 缩放(Scale)
    1. 二维缩放运算：
    ```
    x缩放因子：s1
    y缩放因子：s2
    [x'] = [s1] * [x]
    [y'] = [s2] * [y]
    ```
    2. 二维旋转运算：
    ```
    旋转角度: θ
    R(θ) = [cos(θ), -sin(θ)]
           [sin(θ),  cos(θ)]
    [x'] = R(θ) * [x]
    [y'] = R(θ) * [y]
    ```
    3. 齐次坐标运算：
    ```
    旋转角度: θ
    平移距离: tx, ty
    
    [x']   [cos(θ) -sin(θ) tx] [x]
    [y'] = [sin(θ)  cos(θ) ty] [y]
    [1]    [  0       0     1] [1]
    ```
