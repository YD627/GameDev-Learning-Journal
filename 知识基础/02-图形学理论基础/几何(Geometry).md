## 几何(Geometry)
1. 隐式几何: 定义一个函数描述点的关系
    - 找全一个面的全部点很难
    - 判断一个点不在一个面内简单
2. 显式几何: 定义一个点的坐标
    - 找全一个面的全部点很容易
    - 判断点在表面内还是外比较难
3. 距离函数
    - Blend Distance Function: 用于将多个距离函数组合起来，实现更复杂的几何效果

### Object File Format(.obj)
- 用于存储3D模型的文件格式
- 包含顶点、面、材质等信息
- 可以使用3D软件打开和编辑
- 也可以使用编程语言读取和解析

### 贝塞尔曲线
- 用于描述平滑的曲线
- 递归定义, 每次都是找一个t值, 然后根据t值来计算曲线上的点
- 可以通过控制点来定义曲线的形状
- 例如，圆弧、椭圆、样条曲线等
```
假设有4个点b0, b1, b2, b3
               b30
          b20       b21
     b10       b11       b12
b0        b1        b2        b3
b10(t) = (1-t)*b0 + t*b1
b20(t) = (1-t)*b1 + t*b2 = (1-t)²*b0 + 2*t*(1-t)*b1 + t²*b2
```

### Picewise Bézier Curves（分段贝塞尔曲线）
- 如果两个曲线连接要是连续的话, 则需要满足以下条件:
    - 两个曲线的导数在连接点上相等(共线，长度相等)
- C0连续: 两个曲线在连接点上没有跳转
- C1连续: 两个曲线在连接点上没有跳转, 且在连接点上导数相等

### Loop Subdivision（Loop细分） - 和循环无关
- split each triangle into four
    - 公式: 新的点 = 3/8(A + B) + 1/8(C + D)

- 更新旧的点
    - 公式: （1 - n*u）*旧点 + u*neighbor_position_sum
    - 其中, u是细分参数, n是邻居点的数量, neighbor_position_sum是邻居点的坐标和


### Catmull-Clark Subdivision（Catmull-Clark细分）
- 在非四边形图形中添加奇异点，后就没非四边形了(增加了非4边形数的奇异点)

### 曲面简化(Mesh Simplification)
- edge collapse(边坍缩): 将一条边删除, 并将两个顶点合并为一个新的顶点
    - Quadric Error Metric(四元二次误差度量): 找一个点, 使合并后的顶点与原始顶点的距离最小化
