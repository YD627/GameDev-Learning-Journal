## 你好，窗口
1. 头文件配置
    ```C++
    # include <glad/glad.h>
    # include <GLFW/glfw3.h>
    ```
2. 实例化GLFW窗口
    ```C++
    int main() {
        glfwInit();
        glfwWindowHint(GLFW_VERSION_CONTEXT_MAJOR, 3);
        glfwWindowHint(GLFW_VERSION_CONTEXT_MINOR, 3);
        glfwWindowHint(GLFW_OPENGL_PROFILE, GLFW_OPENGL_CORE_PROFILE);

    }
    ```
    首先，在main中调用glfwInit函数来初始化GLFW，然后我们可以使用glfwWindowHint函数来配置GLFW。
    我们将主版本号(Major)和次版本号(Minor)都设为3。我们同样明确告诉GLFW我们使用的是核心模式(Core-profile)。
3. 创建窗口对象
    ```C++
    GLFWwindow* window = glfwCreateWindow(800，600， “LearnOpenGL”, NULL, NULL);
    if(window == NULL) {
        std::cout << "Failed to create GLFW window" << std::endl;
        glfwTerminate();
        return -1;
    }
    glfwMakeContextCurrent(window);
    ```
    glfwCreateWindow函数需要窗口的宽和高作为它的前两个参数。第三个参数表示这个窗口的名称（标题），这里我们使用"LearnOpenGL"，当然你也可以使用你喜欢的名称。最后两个参数我们暂时忽略。这个函数将会返回一个GLFWwindow对象，我们会在其它的GLFW操作中使用到。
    创建窗口对象，我们检查窗口是否创建成功。如果创建失败，我们打印错误信息并终止GLFW。
    如果窗口创建成功，我们使用glfwMakeContextCurrent函数来将窗口上下文设置为当前上下文。
4. 初始化GLAD
    ```C++
    if(!gladLoadGLLoader((GLADloadproc)glfwGetProcAddress)) {
        std::cout << "Failed to initialize GLAD" << std::endl;
        return -1;
    }
    ```
    GLAD是用来管理OpenGL的函数指针的，所以在调用任何OpenGL的函数之前我们需要初始化GLAD。我们给GLAD传入了用来加载系统相关的OpenGL函数指针地址的函数。GLFW给我们的是glfwGetProcAddress，它根据我们编译的系统定义了正确的函数。
5. 视口
    ```C++
    glViewport(0, 0, 800, 600);
    ```
    glViewport函数用来设置视口的大小。第一个参数是视口的左下角的x坐标，第二个参数是视口的左下角的y坐标，第三个参数是视口的宽度，第四个参数是视口的高度。
    我们将视口设置为窗口的大小，所以第一个参数是0，第二个参数是0，第三个参数是窗口的宽，第四个参数是窗口的高。
6. 对窗口注册一个回调函数(Callback Function)
    ```C++
    void framebuffer_size_callback(GLFWwindow* window, int width, int height) {
        glViewport(0, 0, width, height);
    }
    ```
    这个帧缓冲大小函数需要一个GLFWwindow作为它的第一个参数，以及两个整数表示窗口的新维度。每当窗口改变大小，GLFW会调用这个函数并填充相应的参数供你处理。
    我们还需要注册这个函数，告诉GLFW我们希望每当窗口调整大小的时候调用这个函数：
    ```C++
    glfwSetFramebufferSizeCallback(window, framebuffer_size_callback);
    ```
7. 循环渲染
    ```C++
    while(!glfwWindowShouldClose(window)) {
        glfwPollEvents();
        glfwSwapBuffers(window);
    }
    ```
    - glfwWindowShouldClose用来检测窗口是否关闭，如果窗口关闭，返回true，否则返回false。
    - glfwPollEvents函数用来检查是否有事件发生。
    - glfwSwapBuffers函数用来交换前缓冲区和后缓冲区，从而实现双缓冲。
8. 释放资源
    ```C++
    glfwDestroyWindow(window);
    glfwTerminate();
    ```
    - glfwDestroyWindow函数用来销毁窗口对象。
    - glfwTerminate函数用来终止GLFW，释放所有资源。
9. 输入处理
    ```C++
    void processInput(GLFWwindow* window) {
        if(glfwGetKey(window, GLFW_KEY_ESCAPE) == GLFW_PRESS) {
            glfwSetWindowShouldClose(window, true);
        }
    }
    ```
    这里我们检查用户是否按下了返回键(Esc)（如果没有按下，glfwGetKey将会返回GLFW_RELEASE。如果用户的确按下了返回键，我们将通过使用glfwSetWindowShouldClose把WindowShouldClose属性设置为 true来关闭GLFW。下一次while循环的条件检测将会失败，程序将关闭。
    ```C++
    while(!glfwWindowShouldClose(window)) {
        processInput(window);
        glfwPollEvents();
        glfwSwapBuffers(window);
    }
    ```
    在渲染循环的每一个迭代中调用processInput函数，来检查是否有用户输入。
10. 渲染循环
    ```C++
    glClearColor(0.2f, 0.3f, 0.3f, 1.0f);
    glClear(GL_COLOR_BUFFER_BIT);
    ```
    - glClearColor函数用设置清空屏幕所用的颜色，第一个参数是红色通道，第二个参数是绿色通道，第三个参数是蓝色通道，第四个参数是alpha通道。
    - glClear函数用来清除颜色缓冲区。(可能的缓冲位有GL_COLOR_BUFFER_BIT，GL_DEPTH_BUFFER_BIT和GL_STENCIL_BUFFER_BIT。)
        - GL_COLOR_BUFFER_BIT用来清除颜色缓冲区。
        - GL_DEPTH_BUFFER_BIT用来清除深度缓冲区。
        - GL_STENCIL_BUFFER_BIT用来清除模板缓冲区。
    ```C++
    while(!glfwWindowShouldClose(window)) {
        processInput(window);

        // 渲染指令
        glClearColor(0.2f, 0.3f, 0.3f, 1.0f);
        glClear(GL_COLOR_BUFFER_BIT);

        glfwPollEvents();
        glfwSwapBuffers(window);
    }
    ```
