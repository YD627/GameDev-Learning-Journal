# 插值算法：线性插值与双线性插值详解

## 1. 单线性插值法

### 定义
线性插值法假设在两个已知数据点 `(x₁, y₁)` 和 `(x₂, y₂)` 之间，任意点的值 `y` 可以通过线性关系进行估计。这意味着，我们假设在两个已知点之间，数据的变化是线性的，即可以用一条直线来近似表示这些点之间的数据关系。

### 数学公式
给定两个点 `(x₁, y₁)` 和 `(x₂, y₂)`，在区间 `[x₁, x₂]` 内，对于任意点 `x` 的插值 `y` 为：

```
y = y₁ + (x - x₁) × (y₂ - y₁) / (x₂ - x₁)
```

等价地，可以表示为权重形式：

```
y = (x₂ - x) / (x₂ - x₁) × y₁ + (x - x₁) / (x₂ - x₁) × y₂
```

其中，权重 `t = (x - x₁) / (x₂ - x₁)` 满足 `0 ≤ t ≤ 1`。

### 算法步骤
1. 计算插值权重：`t = (x - x₁) / (x₂ - x₁)`
2. 进行线性插值：`y = y₁ + t × (y₂ - y₁)`
3. 或者使用权重形式：`y = (1 - t) × y₁ + t × y₂`

### 应用场景
1. **数据平滑与填充**：在信号处理、传感器数据处理等领域中，当某些数据缺失或需要平滑时，可以使用单线性插值来估算这些缺失或不平滑的数据点。
2. **图像处理**：在图像处理中，单线性插值常用于图像的缩放、旋转等操作。虽然它可能不如双线性插值或更高阶的插值方法精确，但由于其计算简单，因此在某些对图像质量要求不高的场合仍然得到应用。
3. **物理模拟**：在物理模拟中，单线性插值可以用于估算物体在连续时间内的位置、速度等物理量。
4. **金融分析**：在金融领域，单线性插值可以用于估算股票价格、汇率等金融指标在任意时刻的值。

## 2. 双线性插值法

### 定义
双线性插值（Bilinear Interpolation）是基于线性插值在二维空间上的扩展。它假设在一个矩形区域内，由四个已知点（通常是待插值点的左上、右上、左下和右下角）围成的区域内，数据的变化在两个方向上都是线性的。因此，可以通过在这四个点之间进行两次线性插值（先水平后垂直，或先垂直后水平，结果相同）来估算待插值点的值。

### 数学公式
假设四个角点坐标和值分别为：
- `Q₁₁ = (x₁, y₁)`，值为 `f(Q₁₁)`
- `Q₂₁ = (x₂, y₁)`，值为 `f(Q₂₁)`
- `Q₁₂ = (x₁, y₂)`，值为 `f(Q₁₂)`
- `Q₂₂ = (x₂, y₂)`，值为 `f(Q₂₂)`

需要在点 `P = (x, y)` 处进行插值，其中 `x₁ ≤ x ≤ x₂`，`y₁ ≤ y ≤ y₂`。

**算法步骤：**

1. **第一次线性插值（x方向）**：
   ```
   R₁ = (x₂ - x)/(x₂ - x₁) × f(Q₁₁) + (x - x₁)/(x₂ - x₁) × f(Q₂₁)  // 在y₁处
   R₂ = (x₂ - x)/(x₂ - x₁) × f(Q₁₂) + (x - x₁)/(x₂ - x₁) × f(Q₂₂)  // 在y₂处
   ```

2. **第二次线性插值（y方向）**：
   ```
   f(P) = (y₂ - y)/(y₂ - y₁) × R₁ + (y - y₁)/(y₂ - y₁) × R₂
   ```

**综合公式（一步计算）：**
```
f(x,y) = 1/((x₂-x₁)(y₂-y₁)) × [
    f(Q₁₁)(x₂-x)(y₂-y) + f(Q₂₁)(x-x₁)(y₂-y) +
    f(Q₁₂)(x₂-x)(y-y₁) + f(Q₂₂)(x-x₁)(y-y₁)
]
```

### 关键特性
1. **插值顺序无关性**：先水平后垂直，或先垂直后水平，结果相同。
2. **连续性**：双线性插值在边界上是连续的，但在整个区域内只是C⁰连续（函数值连续，但导数不连续）。
3. **计算复杂度**：只需3次线性插值（2次x方向，1次y方向）即可完成。

### 应用场景
1. **图像缩放**：将图像放大时，为新增的像素点计算颜色值。
2. **纹理映射**：在三维图形渲染中，将二维纹理映射到三维模型表面时，计算非整数纹理坐标处的颜色。
3. **地理信息系统**：在规则网格数据（如海拔、温度）中，估算任意位置的值。
4. **数值模拟**：在有限元分析、计算流体力学等领域，在网格点之间估算物理量的值。

### 注意事项
1. **数据要求**：四个角点应构成矩形区域，且已知点的值应合理。
2. **精度限制**：双线性插值只能重建线性变化，无法精确重建高阶变化。
3. **计算效率**：相比最近邻插值，双线性插值计算量更大，但结果更平滑。

## 3. 线性插值与双线性插值的对比

| 特性 | 单线性插值 | 双线性插值 |
|------|------------|------------|
| **维度** | 一维 | 二维 |
| **输入点数** | 2个点 | 4个点 |
| **计算复杂度** | O(1) | O(1) |
| **连续性** | C⁰连续 | C⁰连续 |
| **应用领域** | 信号处理、一维数据 | 图像处理、二维数据 |
| **平滑性** | 线性平滑 | 双线性平滑 |

---

## 总结

线性插值和双线性插值是计算机图形学、图像处理和科学计算中最基础的插值方法。它们通过简单的数学公式，在已知数据点之间构建连续的估计值，广泛应用于各种需要数据重建和填充的场景。理解这两种插值方法的原理和实现，是学习更高级插值技术（如双三次插值、样条插值等）的重要基础。
